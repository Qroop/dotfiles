## Rofi configuration

This directory contains configuration for the Rofi application launcher.

### `config.rasi`

- Sets the font to `DejaVu Sans Mono 16`.
- Configures Rofi to show icons and use `alacritty` as the terminal for spawned commands.
- Customizes keybindings to match common Vim-style navigation:
  - `Ctrl+h` / `Ctrl+l` for moving the cursor left and right.
  - `Ctrl+j` / `Ctrl+k` for moving up and down through entries.
  - Custom delete and accept-entry bindings.
- Uses the `gruvbox-dark-hard` theme.
