## GTK 3 configuration

This directory contains GTK 3 configuration that affects the look and feel of GTK applications.

### `settings.ini`

- Sets the GTK theme to `HighContrastInverse`.
- Uses the `Adwaita` icon theme, font, and cursor theme.
- Configures toolbar style and icon sizes.
- Enables event and input feedback sounds.
- Enables font antialiasing and hinting with `hintmedium`.
